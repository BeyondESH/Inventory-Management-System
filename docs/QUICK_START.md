# 🚀 食品服务公司管理系统 - 快速使用指南

## 启动方式

### 🔐 方式一：完整启动（推荐）
```bash
python system_launcher.py
```
- 显示登录界面
- 可以选择登录或测试模式
- 登录成功后自动进入主系统

### ⚡ 方式二：直接启动主系统
```bash
python inventory_system.py
```
- 跳过登录直接进入管理系统
- 适合开发测试使用

## 📋 功能模块

### 五大核心模块
1. **📦 库存管理** - 食材库存追踪和预警
2. **🍜 餐食配置** - 菜单管理和价格设定
3. **📋 订单管理** - 订单创建和状态跟踪  
4. **👥 客户管理** - 客户信息和历史记录
5. **💰 财务管理** - 收支分析和利润计算

### 登录注册功能

- 📝 新用户注册

- 🧪 测试模式快速进入
